# 🎰 Cafe Lucky Event - 행운의 추첨기

Streamlit을 이용한 카페 방문 고객용 로또 추첨 이벤트 프로그램입니다. 실감 나는 애니메이션과 이력 관리 기능을 제공합니다.

## 🌟 주요 기능

- **역동적인 추첨 애니메이션**: 회전하는 투명 날개와 무작위로 튀어 오르는 공 효과를 통해 실제 추첨기 같은 생동감을 제공합니다.
- **번호 선택 및 추첨**: 1~45번 중 원하는 번호만 추첨기에 넣거나, 뽑을 공의 개수를 자유롭게 설정할 수 있습니다.
- **수동 결과 입력**: 실제 당첨 번호나 특정 번호를 직접 이력에 추가할 수 있습니다.
- **이력 및 통계**: 추첨된 모든 번호는 자동으로 기록되며, 번호별 출현 빈도를 그래프로 확인할 수 있습니다.
- **데이터 관리**: 이력을 CSV 파일로 다운로드하거나 초기화할 수 있습니다.

## 🛠 설치 및 실행 방법

### 1. 필수 라이브러리 설치
```bash
pip install -r requirements.txt
```

### 2. 로컬에서 실행
```bash
streamlit run lotto_sim.py
```

## 📂 파일 구조

- `lotto_sim.py`: 메인 실행 파일 (Streamlit 기반)
- `requirements.txt`: 필요한 라이브러리 목록
- `background.png`: 추첨기 배경 이미지
- `bounce.wav`: 공이 나올 때 재생되는 효과음
- `lotto_history.txt`: 추첨 이력이 저장되는 텍스트 파일

## 🚀 배포 (Streamlit Community Cloud)

1. 이 저장소를 GitHub에 업로드합니다.
2. Streamlit Cloud에 접속하여 새로운 앱을 만듭니다.
3. 메인 파일 경로(Main file path)를 `lotto_sim.py`로 지정하고 배포합니다.
